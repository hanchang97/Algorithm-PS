- <풀이1>
- 그래프
- dfs를 통해 사이클을 탐색하고 사이클에 포함되는 노드들 파악 
- 사이클이 아닌 정점들 각각 사이클까지의 최단거리를 구하기 위해 bfs 반복

- <풀이2>
- 그래프
- dfs를 통해 사이클을 탐색하고 사이클에 포함되는 노드들 파악 
- 풀이1 과 다르게 bfs 한 번으로 해결
- 사이클에 속하는 하나의 노드에서 bfs 를 시작해서 모든 노드들 까지의 최단 거리를 구함
- 사이클에 속한 노드들은 서로 최단 거리가 0
- 풀이1 보다 빠름
